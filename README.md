# AdventureGame

Simple text-based adventure game written in Python.

## Description

This repository contains a small interactive console game implemented in `adventure_game.py`.

## Requirements

- Python 3.8+

## Run

From the project root run:

```
python adventure_game.py
```

Or on Windows if `python` isn't mapped:

```
py -3 adventure_game.py
```

## Files

- `adventure_game.py` — the game source file.
- `report.txt` — a project report or notes.
